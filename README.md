# VIPSTUDY - Premium Learning Platform

A beautiful glassmorphism design website for VIPSTUDY educational platform.

## Features
- Modern glassmorphism UI design
- Fully responsive layout
- Smooth animations and transitions
- Premium educational apps showcase
- Telegram & WhatsApp integration
- Vercel optimized deployment

## How to Deploy on Vercel

### Method 1: Direct Upload (Easiest)
1. Go to [vercel.com](https://vercel.com)
2. Sign up/login with GitHub
3. Click "New Project"
4. Drag and drop all the files in this folder
5. Click "Deploy"

### Method 2: Using GitHub
1. Create a new GitHub repository
2. Upload all files to the repository
3. Connect the repository to Vercel
4. Deploy automatically

### Method 3: Using Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Login to Vercel
vercel login

# Deploy
vercel

# For production
vercel --prod
